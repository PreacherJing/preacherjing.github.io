使用Hexo+Github快速搭建简洁的个人博客

2017-06-22

通过hexo+github搭建博客简洁快速，以下记录本博客的搭建过程（本文基于mac环境）。

准备工作

安装git

如果你安装过Xcode，那么应该就已经有安装git，否则去官网下载安装，这里需要用git将代码推送到github。

安装nodejs

推荐使用nvm方式安装，方便管理node的不同版本。npm推荐使用淘宝的镜像加速资源访问。

github账号

创建一个github账号，并新增一个username.github.io仓库，username代表你在github的用户名，之后可以通过https://username.github.io访问自己的博客。

安装Hexo

安装hexo

$ npm install -g hexo

可以通过-v命令判断是否安装成功，我这里的版本是3.3.7

$ hexo -v
$ hexo: 3.3.7

新建一个文件夹，比如blog，进入到这个文件夹下执行hexo初始化

$ hexo init

生成静态文件

hexo g (或 hexo generate)

启动服务

$ hexo s (或 hexo server)

启动后时会提示

$ INFO  Start processing
$ INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

我们就可以通过http://localhost:4000/进行本地预览了.

部署至github

blog目录下有一个_config.yml配置文件，最下面可以配置deploy信息。

我是这么配置的，注意type和repo的冒号后面有个空格，另外，不同的hexo版本以下参数也可能不太一样。

deploy:
    type: git
    repo: https://github.com/PreacherJing/preacherjing.github.io.git

安装hexo-deployer-git

$ npm install hexo-deployer-git --save

生成静态代码并部署（我用的是https方式，第一次部署会需要输入github账户名和密码）

$ hexo g
$ hexo d (或 hexo deploy)

此时访问https://username.github.io 就可以看到我们的博客了。

配置个人信息和主题

安装NexT主题，将代码clone到本地，以后主题升级可以直接pull更新。

$ git clone https://github.com/iissnan/hexo-theme-next themes/next

然后在_config.yml中修改个人信息和主题

title: Jing's Blog
author: Jing
language: zh-Hans

theme: next

重新发布，建议每次先使用clean。

hexo clean
hexo g
hexo d

发布新文章

使用命令

hexo new post '文章标题'

当然，直接在blog/source/_post下新建.md文件也可以，所有默认的layout为post的文件都是保存在这个路径下的，但是会没有标题什么的。因为使用命令创建的文件会生成以下默认项，你可以自行添加。

---
title: thisisatest
date: 2017-06-22 15:35:02
tags:
---

删除文章

先通过hexo clean删除包括public的所有文件，然后删除你_post下你想要删除的文件。

hexo g
hexo d

部署即可。
